# SurfShield Releases

This repository hosts the public installers for [SurfShield](https://getsurfshield.com) — privacy software that automatically blurs your screen when someone is looking over your shoulder.

## Download

Visit the [Releases page](https://github.com/jamielamont333-ai/SurfShield-releases/releases) for the latest version, or get the recommended installer from [getsurfshield.com](https://getsurfshield.com).
